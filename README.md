## Useful Docker Commands

```bash
docker compose up -d
```

Start all services.

```bash
docker compose down
```

Stop all services.

```bash
docker compose logs -f
```

View logs.

```bash
docker ps
```

List running containers.

```bash
docker compose build
```

Rebuild images.

```bash
docker volume ls
```

List Docker volumes.

```bash
docker network ls
```

List Docker networks.
